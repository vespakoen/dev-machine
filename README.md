My Dev Machine
===========

Optimized for 64 bit debian based systems and comes with:

- Ubuntu desktop (this will be removed soon)
- GNOME shell
- Xmonad
- FlatStudio theme
- Nitrux icons
- Virtualbox
- Vagrant (with salty-vagrant plugin)
- Sublime Text
- ZSH + Oh My ZSH + Ubuntu Mono Powerline font
